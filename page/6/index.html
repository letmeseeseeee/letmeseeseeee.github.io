PHP 变量规则：

变量以 $ 符号开始，后面跟着变量的名称
变量名必须以字母或者下划线字符开始
变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
变量名不能包含空格
变量名是区分大小写的（$y 和 $Y 是两个不同的变量）

analyse the code

$_page = mb_substr(
$page,
0,
mb_strpos($page . ‘?’, ‘?’)
);
使用了 PHP 的 mb_substr() 和 mb_strpos() 函数来截取字符串。这些函数是用于处理多字节字符的版本，可以正确处理包含非 ASCII 字符的字符串。
mb_substr() 函数用于返回一个字符串的子串。它接受三个参数：原始字符串 $page，子串的起始位置和子串的长度。在这段代码中，起始位置是 0，长度是通过 mb_strpos() 函数计算得出的。
mb_strpos() 函数用于在字符串中查找指定字符或子字符串的位置。它接受两个参数：要搜索的字符串和要查找的字符或子字符串。在这段代码中，它用于查找 $page 字符串中第一个出现的 ‘?’ 字符的位置。
通过这段代码，$_page 变量将被赋值为 $page 字符串中 ‘?’ 字符之前的子串。

综上所述，给定的代码通过使用 PHP 的多字节字符串函数来截取 $page 字符串中 ‘?’ 字符之前的子串。
