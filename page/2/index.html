ANALYSIS

We have a class named X and the private variable is int a

and we have a public member function named showX() and constructor X(int x)

this function is used to initialize the value of a

what’s more, we also defined a class named Y, It inherits privately from class X

It has a private member variable named

“int b”, and the public member is function showY() and a consrtuctor Y(int x, int y)

This constructor initialize the Inherited members by calling the base class X’s constructor

call(调用) constructor(构造函数)

Written like “Y(int x, int y):X(x)”, here the X(x) is calling the base class X’s constructor

We also have a class Z inherits publicly from class Y

It has a private member variable named “int b”, with public member function The constructor

Z (int x, int y, int z) initializes the inherited members by calling the constructor of the base class Y.

and the function showZ() in the function showZ() we use showY() to call the varible b in the class Y

change：the difference between Y inherit X and Z inherit Y is one is private and the other is public

so when in the definition of showY() in class Y we have to use an showX() to call the variable in the

class X, and there is a way to call the function in its Parent class. That is X::showX() to define the

function showX() in the classs Y

It’s easier when we have to rewrite the definition of a same function in a child class
