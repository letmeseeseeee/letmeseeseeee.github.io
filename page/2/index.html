首先，代码中定义了一个名为authUrl的数组，包含了一些需要进行验证的URL。

然后，使用request.interceptors.request.use()方法注册了一个请求拦截器。该拦截器会在每个请求发送之前执行给定的回调函数。

在回调函数中，首先通过检查config.url是否在authUrl数组中来确定是否需要进行验证。

如果需要验证，代码会执行以下步骤：

从本地存储中获取名为”login”的数据，并将其解析为loginMsg对象，该对象包含了访问令牌(access token)和刷新令牌(refresh token)的信息。

检查访问令牌(access token)是否过期。通过解码令牌中的信息，比较其过期时间(expiration time)与当前时间，如果过期则进行进一步处理。

如果访问令牌过期，继续检查刷新令牌(refresh token)是否过期。同样，通过解码刷新令牌中的信息，比较其过期时间与当前时间，如果过期则执行相应的处理。

如果刷新令牌未过期，向服务器发送一个/refresh/的POST请求，传递刷新令牌作为参数。根据返回的响应，更新本地存储中的登录信息。

如果验证通过，将访问令牌(access token)添加到请求的头部中，使用Bearer身份验证方式。

如果验证未通过，通过this.$router.push()方法跳转到指定的登录页面。

最后，回调函数需要返回config对象或一个Promise对象，以便允许请求继续发送，或者中止请求。

代码中的config对象并非JavaScript语言的原生对象，而是特定框架或库（如Axios）提供的请求配置对象。

在请求拦截器中，config参数是由Axios库提供的，用于配置发送的HTTP请求的对象。它包含了请求的URL、请求方法、请求头部、请求体等信息。

具体来说，config对象可能包含以下属性：

url: 请求的URL地址。
method: 请求的方法，如GET、POST等。
headers: 请求的头部信息，包括Content-Type、Authorization等。
params: GET请求的查询参数。
data: POST请求的请求体数据。
timeout: 请求的超时时间。
responseType: 期望的响应数据类型。

在拦截器的回调函数中，可以通过访问和修改config对象来对请求进行处理和配置。例如，可以添加请求头部、修改URL、设置请求超时时间等。

config.headers.Authorization是一个HTTP请求头部字段，用于携带身份验证凭据或授权信息。

DAY 24 c++

#include
using namespace std;

class Shape {
public:
virtual double getArea() const = 0;
virtual double getPerimeter() const = 0;
};

class Circle : public Shape {
private:
double radius;

public:
Circle(double r) : radius(r) {}

double getArea() const override {
    return 3.14159 * radius * radius;
}

double getPerimeter() const override {
    return 2 * 3.14159 * radius;
}

};

class Square : public Shape {
private:
double side;

public:
Square(double s) : side(s) {}

double getArea() const override {
    return side * side;
}

double getPerimeter() const override {
    return 4 * side;
}

};

class Rectangle : public Shape {
private:
double length;
double width;

public:
Rectangle(double l, double w) : length(l), width(w) {}

double getArea() const override {
    return length * width;
}

double getPerimeter() const override {
    return 2 * (length + width);
}

};

int main() {
const int numShapes = 3;
Shape* shapes[numShapes];

shapes[0] = new Circle(5.0);
shapes[1] = new Square(4.0);
shapes[2] = new Rectangle(3.0, 6.0);

double totalArea = 0.0;
for (int i = 0; i < numShapes; i++) {
    totalArea += shapes[i]->getArea();
}

cout << "所有图形的面积之和: " << totalArea << endl;

// 释放动态分配的内存
for (int i = 0; i < numShapes; i++) {
    delete shapes[i];
}

return 0;

}

$whitelist = [“source”=>”source.php”,”hint”=>”hint.php”];

这串代码是一个PHP数组赋值操作，创建了一个名为$whitelist的关联数组，包含两个键值对。数组的键是字符串，分别为”source”和”hint”，对应的值分别是”source.php”和”hint.php”。这意味着在

$whitelist数组中，”source”键关联的值是”source.php”，”hint”键关联的值是”hint.php”。这种数组结构可以用来存储和访问相关的键值对数据。

if (! isset($page) || !is_string($page)) {
echo “you can’t see it”;
return false;
}

一个条件语句。它首先使用isset()函数检查变量$page是否已经设置。如果$page未设置（即不存在或为null），条件成立。接下来，使用is_string()函数检查$page的类型是否为字符串。如果$page不是字符串类型，条件同样成立。

如果以上两个条件之一成立，代码块内的语句将被执行。在这种情况下，会输出字符串”you can’t see it”。然后，使用return语句返回false，表示在这种情况下函数或代码的执行将会终止。

PHP 变量规则：

变量以 $ 符号开始，后面跟着变量的名称
变量名必须以字母或者下划线字符开始
变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
变量名不能包含空格
变量名是区分大小写的（$y 和 $Y 是两个不同的变量）
