实验八的重写说明

为什么要使用虚基类?

使用虚基类的目的是避免出现多次继承导致数据冗余，可以确保在派生类中只有一个共享的基类的子对象

在给定的代码中，PartTime_Graduate_Student 类继承了 Employee 和 Student 两个类

这两个类Employee和Student都继承了person类，如果没有使用虚基类，PartTime_Graduate_Student类将会包含两个独立person子对象，这两个冗余数据分别来自于Employee和Student类的继承

这样就会导致两个person子对象中的数据比如说：name、age 和 sex重复存储浪费内存空间，

通过使用虚基类，Person成为虚基类，派生类 PartTime_Graduate_Student 只包含一个 Person 子对象

只有一个共享的Person对象被创建。派生类PartTime_Graduate_Student只包含一个Person对象,所以称Person为子对象

void gotoxy(int x, int y) {//位置函数(行为x 列为y)
COORD pos;
pos.X=2*y;
pos.Y=x;
SetConsoleCursorPosition (GetStdHandle (STD_OUTPUT_HANDLE),pos);
}
上面的代码是一个用于设置光标位置的函数。它使用了Windows系统的Console API来设置光标的位置。

函数定义了一个COORD结构类型的变化量pos，用于存储光标记的位置设置。

该任数使用了Windows系统的控制台数来实现了标准位置的设置。COORD是一个结构体，它包含了控制台屏幕的行数和列表信息。在任数内部，首先创建一个类型的变量，然后COORD将pos参数x和y分配值给pos.Y和pos.X。在这里，行数对应参数x，列数对应参数y。最后，调用SetConsoleCursorPosition任意数来设置控制台灯标的位置，使之转移动到pos指定的行位置。

COORD是一个库的自带结构体，包含控制台屏幕的行数和列表信息

step by step

1.A color function
void color (int a) {//颜色函数

}
define a color program

use a console API SetConsoleTextAttribute to set the color of the text the console output

variable a Represents a color code which is used to designate the The foreground and background colors of the text

With this code, you can change the appearance of the text by passing different color codes.

句柄通常是一个整数值，用于唯一标识操作系统或应用程序中的资源或对象，例如文件、窗口、进程等。

GetStdHandle函数返回标准输入、输出或错误设备的句柄，也就是获得输入、输出/错误的屏幕缓冲区的句柄。通过获得这些句柄，程序可以对其进行进一步操作或设置。

GetStdHandle (STD_OUTPUT_HANDLE)是一个Windows API函数，用于获取标准输出的句柄

GetStdHandle函数接受一个参数，指定所需的标准句柄类型，其中STD_OUTPUT_HANDLE表示标准输出句柄。该句柄通常用于在控制台应用程序中进行标准输出，即向控制台窗口输出文本或其他数据。函数会返回与该标准句柄类型相对应的句柄值

SetConsoleTextAttribute是Windows操作系统中的一个函数，用于设置控制台窗口中文本的颜色和背景色。

该函数接受一个句柄参数，另外，还需要指定一个由字符和背景色组成的参数，其中高位字节表示背景色，低位字节表示文本颜色。

对于 SetConsoleTextAttribute (GetStdHandle (STD_OUTPUT_HANDLE), a); ，GetStdHandle 函数用于获取标准输出设备的句柄。

这里SetConsoleTextAttribute函数用于设定文本属性，接受两个值，一个句柄值，一个参数a，a的具体值决定了文本颜色和背景颜色

总的来说这段代码的作用是设定文本颜色和背景颜色
