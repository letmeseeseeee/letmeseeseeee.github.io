反思

1.没有使用析构函数，并且基类的析构函数也要加virtual关键词

这是因为将~Shape()声明为虚拟析构函数是为了支持多态性。当基类指针指向派生类对象时，使用虚拟析构函数可以确保通过基类指针正确销毁派生类对象。

在C++中，当派生类对象通过基类指针进行删除时，如果基类的析构函数不是虚拟的，则只会调用基类的析构函数，而不会调用派生类的析构函数。这可能导致内存泄漏和未定义的行为。

因此，为了正确地释放派生类的资源，应将基类的析构函数声明为虚拟的。

2.构造函数很不熟练有两种命名形式

class Circle:public Shape{
private:
float radius;

public:
Circle(float rad){
radius=rad;
}
~Circle(){}
};

或

class Circle:public Shape{
private:
float radius;

public:
Circle(float rad):rad(radius){
radius=rad;
}
~Circle(){}
};

如果没有私有成员的话什么都不用写但仍然需要有构造函数

class Circle:public Shape{
public:
Circle(float rad):rad(radius){
radius=rad;
}
~Circle(){}
};
3.在main函数中用Shape*example 而不是用Shape example 是为了使用指针的多态性质

基类指针Shape* example可以指向派生类的对象，从而通过基类指针调用派生类的虚函数，实现多态性的效果

由此可见，多态性在遍历元素对象方面有着无与伦比的优越性

4.增强理解：多态性与new运算符

使用new运算符可以创建对象的实例。在许多面向对象编程语言中，我们可以使用new运算符来实例化一个类，并创建该类的对象。通过实例化一个类，我们可以使用该类的属性和方法。

多态性的概念使得我们可以使用父类的引用来引用子类的对象，并根据实际的对象类型调用相应的方法。

5.如下，由于example是一个指针，所以需要在nihao前面加上&

6.此外，在使用example调用countArea()函数时，应使用箭头操作符（->）而不是点操作符（.），因为example是一个指针。

原因是example是一个指向Shape对象的指针，而不是一个Shape对象本身。箭头操作符用于通过指针访问成员函数。

7.在使用example调用countArea()函数之前，需要检查example是否为nullptr，以避免空指针解引用错误。

if (example != nullptr) {

}

实验八的重写说明

为什么要使用虚基类?

使用虚基类的目的是避免出现多次继承导致数据冗余，可以确保在派生类中只有一个共享的基类的子对象

在给定的代码中，PartTime_Graduate_Student 类继承了 Employee 和 Student 两个类

这两个类Employee和Student都继承了person类，如果没有使用虚基类，PartTime_Graduate_Student类将会包含两个独立person子对象，这两个冗余数据分别来自于Employee和Student类的继承

这样就会导致两个person子对象中的数据比如说：name、age 和 sex重复存储浪费内存空间，

通过使用虚基类，Person成为虚基类，派生类 PartTime_Graduate_Student 只包含一个 Person 子对象

只有一个共享的Person对象被创建。派生类PartTime_Graduate_Student只包含一个Person对象,所以称Person为子对象

句柄通常是一个整数值，用于唯一标识操作系统或应用程序中的资源或对象，例如文件、窗口、进程等。

GetStdHandle函数返回标准输入、输出或错误设备的句柄，也就是获得输入、输出/错误的屏幕缓冲区的句柄。通过获得这些句柄，程序可以对其进行进一步操作或设置。

GetStdHandle (STD_OUTPUT_HANDLE)是一个Windows API函数，用于获取标准输出的句柄

GetStdHandle函数接受一个参数，指定所需的标准句柄类型，其中STD_OUTPUT_HANDLE表示标准输出句柄。该句柄通常用于在控制台应用程序中进行标准输出，即向控制台窗口输出文本或其他数据。函数会返回与该标准句柄类型相对应的句柄值

SetConsoleTextAttribute是Windows操作系统中的一个函数，用于设置控制台窗口中文本的颜色和背景色。

对于 SetConsoleTextAttribute (GetStdHandle (STD_OUTPUT_HANDLE), a); ，GetStdHandle 函数用于获取标准输出设备的句柄。

这里SetConsoleTextAttribute函数用于设定文本属性，接受两个值，一个句柄值，一个参数a，a的具体值决定了文本颜色和背景颜色
