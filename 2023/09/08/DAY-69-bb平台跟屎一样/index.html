DAY 69 bb平台跟屎一样

华为云的代码托管和bb平台就跟屎一样，

华为云平台看似很简洁，但是指引这么少真的浪费时间，降低效率，烂

bb平台主打一个进不去，也是烂，一直搁那等待模块加载，等待等待等你吗呢

关于idea项目推送到华为云上，用https方法最好是修改一下密码不要用华为云登录密码

原因是我一改就能push了

commit之后git push -u origin master -f

如果有提示remote: Codehub: You are not allowed to force push code to a protected branch on this project.

那么在设置里面把保护分支中的master给删掉就行了

华为云强push之后居然是所有东西都进行覆盖？真是坨屎

github显然不会有这种问题

傻逼bb平台做成这样就不要做啊

可能抛出的异常 throws IOException

挑战十分钟写完作业四，作为铺垫已经铺垫完了

挑战失败，远超十分钟

对于form标签，如果用了form标签，input是submit的话，那么就会自动提交，不需要再写onclick了，如果写了onclick相应的
函数，那么就会先执行onclick函数，然后再提交，如果onclick函数返回false，那么就不会提交

为什么没有返回正常的 response.getWriter().write(“我爱你一万年”)文本值

是因为没有写 response.setCharacterEncoding(“UTF-8”);所以无法调回正常值

对于JPA与JDBC，两者的区别在于jdbc需要手动编写sql语句，而JPA不需要，JPA是自动根据方法名生成sql语句，但是JPA的sql语句

是固定的，无法修改，所以只能修改实体类，修改实体类之后，需要重新生成实体类，重新生成实体类之后，需要重新生成数据库表

由于JPA的sql语句是框架自动生成的，所以比较固定，不如jdbc的精度高
