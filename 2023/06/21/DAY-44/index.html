DAY 44

分别定义Person类、Employee类、Student类以及在职研
究生类PartTime_Graduate_Student，并且画出相应的继承层
次图

显然Person是基类

而student和Employee在第二层

PartTime_Graduate_Student在第三层

这里用到虚基类的知识，第三层是在职研究生既是Employee，也是student

#include

using namespace std;

class Person{
private:
string name;
char sex;
public:
Person(string a,char s):name(a),sex(s){
}
virtual ~Person{

}
virtual void info(){
cout << "名字是" << name ;
cout << "性别是" << sex;
}

};
class student:virtual public Person{
private:
int number;
int grade;
public:
student(int a,int g ):number(a),grade(g){
}
~student(){
}
void info(){
Person.info();
cout << “学号是” << number<< endl;
cout << “年级是” << grade <<endl;
}
};
class Employee:virtual public Person{
private:
int worknum;
double wage;
string job;
public:
Employee(int a,double b , string c):worknum(a),wage(b),job(c){
}
~Employee(){
}
void info (){
Person.info();
cout << “工号是” <<worknum<< endl;
cout << “工资是” <<wage <<endl;
cout << “工作是” <<job << endl ;
}

};
class PartTime_Graduate_Student:public Employee,public student{
private:

public:
void info (){
student.info();
Employee.info();

}

};
int main()
{
PartTime_Graduate_Student nihao(“李伯辉”,”男”,19,1,1,10000,”程序员”);
nihao.info();

return 0;

}

首先，在派生类引用基类的成员函数的时候需要用::而不是.

其次在两个派生类的构造函数中初始化错误，应该引用虚基类的构造函数

并且在最小的那个派生类中的构造函数初始化成员数值也出现了错误

PartTime_Graduate_Student(string a, char b ,int d ,int e ,int f ,double g ,string h):Person(a,b),student(d,e),Employee(f,g,h)
{
}
上面是重写的一个派生类，但仍然有错误

再用student和Employee进行构造函数对数值进行初始化的时候要把之前的虚基类中的两个私有成员数值也加上

PartTime_Graduate_Student(string a, char b ,int d ,int e ,int f ,double g ,string h):Person(a,b),student(a,b,d,e),Employee(a,b,f,g,h)
{
}
修改好之后的函数

并且，由于b是char类型所以男应该用单引号括‘c’，而不是双引号”c”

一开始输出错误了，但是加上endl又正常输出了不知道为什么

如果不使用 << endl，输出的内容将保持在输出缓冲区中，直到达到缓冲区刷新的条件（例如缓冲区填满或遇到换行符等），才会将内容显示在终端上。因此，如果不加 << endl，输出可能会被缓冲而不立即显示。

派生类要用虚基类的成员时，构造函数中要把基类的成员也写上
