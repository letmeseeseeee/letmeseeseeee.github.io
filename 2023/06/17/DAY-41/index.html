// 定义一个函数对象（Functor）
class MultiplyBy {
private:
int factor;
public:
MultiplyBy(int factor) : factor(factor) {}

// 重载函数调用运算符
int operator()(int value) const {
    return value * factor;
}

};

int main() {
std::vector numbers = {1, 2, 3, 4, 5};
int factor = 2;

// 创建函数对象
MultiplyBy multiply(factor);

// 使用函数对象作为算法参数，而不是将函数所执行的运算作为算法的一部分
std::transform(numbers.begin(), numbers.end(), numbers.begin(), multiply);

// 打印结果
for (const auto& number : numbers) {
    std::cout << number << " ";
}
std::cout << std::endl;

return 0;

}
在 MultiplyBy multiply(factor);
这一步的时候，multiply这个对象的私有成员factor的值被设置为2

std::transform(numbers.begin(), numbers.end(), numbers.begin(), multiply);

这一步的时候执行算法numbers向量中的每一个元素作为参数传递给函数对象的函数调用运算符

算法（Algorithms）：STL 提供了大量的算法，如排序、查找、遍历、变换等，可用于对容器中的数据进行操作和处理。这些算法是通用的，适用于各种容器类型，而无需开发者自己实现。
