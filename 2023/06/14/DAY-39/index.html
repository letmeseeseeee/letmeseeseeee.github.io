反思

1.没有使用析构函数，并且基类的析构函数也要加virtual关键词

这是因为将~Shape()声明为虚拟析构函数是为了支持多态性。当基类指针指向派生类对象时，使用虚拟析构函数可以确保通过基类指针正确销毁派生类对象。

在C++中，当派生类对象通过基类指针进行删除时，如果基类的析构函数不是虚拟的，则只会调用基类的析构函数，而不会调用派生类的析构函数。这可能导致内存泄漏和未定义的行为。

因此，为了正确地释放派生类的资源，应将基类的析构函数声明为虚拟的。

2.构造函数很不熟练有两种命名形式

class Circle:public Shape{
private:
float radius;

public:
Circle(float rad){
radius=rad;
}
~Circle(){}
};

或

class Circle:public Shape{
private:
float radius;

public:
Circle(float rad):rad(radius){
radius=rad;
}
~Circle(){}
};

如果没有私有成员的话什么都不用写但仍然需要有构造函数

class Circle:public Shape{
public:
Circle(float rad):rad(radius){
radius=rad;
}
~Circle(){}
};
3.在main函数中用Shape*example 而不是用Shape example 是为了使用指针的多态性质

基类指针Shape* example可以指向派生类的对象，从而通过基类指针调用派生类的虚函数，实现多态性的效果

由此可见，多态性在遍历元素对象方面有着无与伦比的优越性

4.增强理解：多态性与new运算符

使用new运算符可以创建对象的实例。在许多面向对象编程语言中，我们可以使用new运算符来实例化一个类，并创建该类的对象。通过实例化一个类，我们可以使用该类的属性和方法。

多态性的概念使得我们可以使用父类的引用来引用子类的对象，并根据实际的对象类型调用相应的方法。

5.如下，由于example是一个指针，所以需要在nihao前面加上&
Shapeexample;
Circle nihao(3.4);
example= nihao;
改为
Shapeexample;
Circle nihao(3.4);
example= &nihao;

6.此外，在使用example调用countArea()函数时，应使用箭头操作符（->）而不是点操作符（.），因为example是一个指针。

原因是example是一个指向Shape对象的指针，而不是一个Shape对象本身。箭头操作符用于通过指针访问成员函数。

7.在使用example调用countArea()函数之前，需要检查example是否为nullptr，以避免空指针解引用错误。

if (example != nullptr) {
cout << “面积是” << example->countArea();
}
8.最后一定要进行delete的操作

修改之后的代码为#include
using namespace std;

const float Pi = 3.14159;

class Shape {
public:
Shape() {}
virtual ~Shape() {}
virtual float countArea() {}
virtual float countPerimeter() {}
};

class Circle : public Shape {
private:
float radius;

public:
Circle(float rad) {
radius = rad;
}
~Circle() {
}
float countArea() {
return Pi * radius * radius;
}
float countPerimeter() {
return 2 * Pi * radius;
}
};

class Square : public Shape {
private:
float side;

public:
Square(float sid) {
side = sid;
}
~Square() {
}
float countArea() {
return side * side;
}
float countPerimeter() {
return side * 4;
}
};

class Rectangle : public Shape {
private:
float width;
float length;

public:
Rectangle(float lon, float wid) {
width = wid;
length = lon;
}
~Rectangle() {
}
float countArea() {
return length * width;
}
float countPerimeter() {
return (length + width) * 2;
}
};

int main() {
Shape* example;
Circle nihao(3.4);
example = &nihao;
if (example != nullptr) {
cout << “面积是” << example->countArea();
}
example = new Rectangle(5, 7);
if (example != nullptr) {
cout << “面积是” << example->countArea();
}
delete example;
return 0;
}
