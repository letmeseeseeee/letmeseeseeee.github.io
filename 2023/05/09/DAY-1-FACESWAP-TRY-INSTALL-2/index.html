DAY 1 FACESWAP TRY INSTALL(2)

ILLUSTRATION: My laptop Configuration is NVIDIA GeForce RTX 3050 Laptop GPU

FIRST STEP:

INSTALLTION

install the software anaconda and choose the default option and if you have

the same configuration like me you shall choose the NVIDIA GPU

SECOND STEP:

click environment and create a environment named whatever you want, I name it

faceswap. Just like what I am doing.Hit the “>” arrow next to your faceswap

environment and select “Open Terminal”

THIRD STEP:

Now we will go on with The form of the command line.

Most of the steps are followed by the install.md file.

Get the faceswap repo by typing: “git clone –depth 1 https://github.com/“

deepfakes/faceswap.git

PS: if you don’t want to use the default path like in the c drive, you can

create a folder and named it whatever you like and use command “cd /d path”

to get in the path you want.

FOURTH STEP: Easy install

Enter the command and follow the prompts:python setup.py

If you have issues/errors follow the Manual install steps below.

I don’t know if anyone can do it just for one time. I just have some annoying

errors.

FIFTH STEP: Manully install

Install tkinter (required for the GUI) by typing: conda install tk

For Nvidia GPU users: pip install -r ./requirements/requirements_nvidia.txt

input the command above.

SIXTH STEP: Running faceswap

Enter the faceswap folder: cd faceswap

Enter the following to see the list of commands: or enter to launch the

GUIpython faceswap.py -hpython faceswap.py gui

SEVENTH STEP: Solve problems

if you get the error that prompt you have some module you don’t successful.

you shall use the command to solve.You must enter the correct syntax format can this command work.

pip install ModuleName -i http://pypi.douban.com/simple/ –trusted-host pypi.douban

ERROR: you don’t have tensorflow.

conda list

to check if you have tensorflow

conda install tensorflow==2.10.0

to install the correct tensor flow

and you will see the faceswap gui initiate
