实验七：运算符重载
实验目的：
（1）掌握运算符函数重载为成员函数的方法
（2）掌握运算符函数重载为友元函数的方法
（3）掌握不同类型数据间的转换方法
实验内容：
假设square_matrix是n阶整型方阵，请实现下列运算：
（1）cin>> square_matrix
（2）cout<< square_matrix
（3）IntMatrix(n)生成一个n阶单位矩阵
（4）square_matrix+IntMatrix(n)
（5）square_matrix_A=square_matrix_B

create a Class named IntMatric.

CONSTRUCTOR

Using constructor in the class as Member functions to create a square-matrix and a identity-matrix.

First: a constructor with parameters
IntMatrix(int r,int co) : row(r),column(co) {
// 分配内存空间
//这里构造函数后接一个冒号：size（n）的意思是把n的值赋值给size
matrix = new int*[row];
//在构造函数中分配内存空间来存储矩阵的元素。
//这里通过new int*[size]创建一个大小为size的int指针数组matrix，表示矩阵的行。
//matrix = new int *[size]表示矩阵的行数为size行
for (int i = 0; i < row; i++) {
matrix[i] = new int[column];
// 初始化元素为0
//matrix本身是一个指针数组，这一步的目的是给矩阵的每一行分配内存
//即每一个matrix中的元素都是指向一个整型数组的
for (int j = 0; j < column; j++) {
matrix[i][j] = 0;
}
}
}

IntMatrix(int n) : row(n),column(n){
            matrix = new int*[n];
    //在构造函数中分配内存空间来存储矩阵的元素。
    //这里通过new int*[size]创建一个大小为size的int指针数组matrix，表示矩阵的行。
    //matrix = new int *[size]表示矩阵的行数为size行
    for (int i = 0; i < n; i++) {
        matrix[i] = new int[n];

        //matrix本身是一个指针数组，这一步的目的是给矩阵的每一行分配内存
        //即每一个matrix中的元素都是指向一个整型数组的
        for (int j = 0; j < n; j++) {
            matrix[i][j] = 0;

        matrix[i][i] = 1;
    }
}

Operator functions are overloaded as member functions

IntMatrix operator+(const IntMatrix& other) const {
IntMatrix result(row,column);
//创建一个名为result类型为IntMatrix的对象
for (int i = 0; i < row ; i++) {
for (int j = 0; j < column; j++) {
//result 对应矩阵的每个位置上面的元素
result.matrix[i][j] = matrix[i][j] + other.matrix[i][j];
}
}
return result;
}

friend ostream& operator<<(ostream& output, const IntMatrix& matrix) {
for (int i = 0; i < matrix.row; i++) {
for (int j = 0; j < matrix.column; j++) {
output << matrix.matrix[i][j] << ‘ ‘;
}
//按照正确的格式输出一个矩阵，由于是方阵所以size既是行数也是列数
output << endl;
}
return output;
}

ostream 是输出流的类，用于向输出设备写入数据，这段代码的作用是将矩阵的元素值输出到输出流中

‘<<’运算符重载为友元函数

friend istream& operator>>(istream& input, IntMatrix& matrix) {
for (int i = 0; i < matrix.row; i++) {
for (int j = 0; j < matrix.column; j++) {
input >> matrix.matrix[i][j];
}
}
return input;
}
key word: friend，友元关键词 istream’输入流的类’，用于从输入设备读取数据

istream& input是一个参数声明，它指示该函数接受一个输入流对象作为输入参数

即表示输入流对象引用，用于从输入流中读取数据并且在函数中修改输入流对象并且返回该对象
