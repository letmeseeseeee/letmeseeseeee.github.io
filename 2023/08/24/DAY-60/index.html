题目是
1.创建一个Person类，包含id,name两个属性，添加2个构造函数，重写toString（）方法返回id和name的值，同时包含display方法，用于显示id和name的值；包含静态的daffodilThree方法，用于循环打印100-999之间的水仙花数

2.创建一个测试类TestPerson

实例化Person类对象，并调用其display方法。

打印水仙花数

一个名字是TestPerson.java 装main

一个名字是Person.java 装Person类

Person类中有两个基本属性

代码如下
TestPerson.java
import java.util.Scanner;

public class TestPerson {
    public static void main(String[] args) {
        Person person=new Person(1,"张三");
        person.display();
        daffodilThree();
    }
    public static void daffodilThree(){
        for(int i=100;i<1000;i++){
            int a=i/100;
            int b=i/10%10;
            int c=i%10;
            if(i==a*a*a+b*b*b+c*c*c){
                System.out.println(i);
            }
        }
    }
}
Person.java
public class Person {
    int id;
    String name;
    public Person(){
        //无参构造函数
    }

    public Person(int id,String name){
        this.id=id;
        this.name=name;
        //有参构造函数
    }
    public String toString(){
        return "id:"+this.id+" name:"+this.name;
    }
    public void display(){
        System.out.println(this.toString());
    }
}
